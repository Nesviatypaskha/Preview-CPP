## Prerequisites:
- **Visual Studio** 2017, 2019, 2022
- **.NET Framework** 4.7.2+
- **MetaOutput** extension

---

## Dependencies:
- Data visualization is based on [MetaOutput](https://www.metaoutput.net/)
- Data extraction is based on [LibClang](https://clang.llvm.org)
